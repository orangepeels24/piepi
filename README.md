# Piepi

A pip package manager with a UI. Browse, install, and manage Python packages without touching the terminal.

---

## Requirements

- Node.js 18+
- Python 3.8+

## Install

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## What it does

Search 755,000+ PyPI packages, install with one click, manage multiple Python interpreters, track installed packages across environments.

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `⌘K` | Command palette |
| `/` | Focus search |
| `j` / `k` | Navigate list |
| `Enter` | Open detail |
| `i` | Install focused |
| `f` | Favorite |
| `⌘Z` | Undo |
| `1–4` | Switch views |
| `Esc` | Close / back |
